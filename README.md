C# instance of SubBot

## Running
1. Download or clone.
2. Add `appsecrets.json` file (with "Copy always" or "Copy if newer" for **Copy to Output Directory**).
3. Populate with secrets. See [appsecrets-example.json](appsecrets-example.json) for example.
4. Build and run.
